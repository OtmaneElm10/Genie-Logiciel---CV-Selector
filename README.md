# Mini CV Selector

## Overview
Mini CV Selector is a small Java tool to screen resumes based on skills, experience, and additional criteria while reducing biases and discrimination. It helps identify hidden talent and ensures a fair selection process.

## Features
- Filter candidates by skills and experience
- Reduce bias in the selection process
- Support for YAML resume files
- Customizable scoring criteria

## Technology
- Java
- YAML input for candidate resumes
- Simple rule-based scoring system

realised by Otmane El Mouki and Nedim Trad, project uploaded from the University Gitlab.
